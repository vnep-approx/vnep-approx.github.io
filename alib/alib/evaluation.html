Main function for evaluation, creating plots and saving them in a specific directory hierarchy. Each metric specification and each axes specification combination will be an heatmap plot. Additionally permutation of length 3 with all scenarioparametergeneration will be generated and fixed for plotting. This leads to the directory hierarchy. For examples number_of_req_25 will be the directory which containing plots where parameter number_of_req is fixed to 25. ECDF and scatter plots will be also plotted as default.

Parameters:
  • dc_baseline – unpickled datacontainer of baseline experiments (e.g. MIP)
  • dc_randround – unpickled datacontainer of randomized rounding experiments
  • metric_specifications – metric specifications used by plots (Default value = global_metric_specifications)
  • axes_specifications – each axes specifications used as x/y axis in plots (Default value = global_metric_specifications)
  • show_plot – show plot after generating it (Default value = False)
  • save_plot – save plots to given structure (Default value = False)
  • do_not_generate_plot_if_file_exists – False if you want to overwrite existing plots (Default value = False)
  • forbidden_scenario_ids – List of scenario ids which should NOT be included in evaluation (Default value = None)
  • plot_ecdf – plot ECDF plots (Default = True)
  • plot_scatter – plot scatter plots (Default = True)
  • maxdepthfilter – Permutation legnth for scenarioparametergeneration (Default = 3)
